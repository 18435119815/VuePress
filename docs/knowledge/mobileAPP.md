# 移动端开发

## 分类
<img src="/assets/img/mobileaAPP.jpg" width="500" height="auto" align="middle" />

类似于uniapp、微信小程序这种的称为webview,通过Javascript bridge与后台进行交互

#### 优点
- 一套代码就够了，而且用的是web前端语言
#### 缺点
- 中间有Javascript bridge桥梁，性能大大的受到影响


## 微信小程序

### 文件结构

- .json  json配置文件  
- .wxml  模板文件(相当于html)
- .wxss  样式文件(相当于css)
- .js    json配置文件
